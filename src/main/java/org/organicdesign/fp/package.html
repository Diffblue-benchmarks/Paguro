Clojure's immutable collections made type safe, an immutable transformation builder, a data definition language, and other tools for writing functional Java.

Highlights:
  • Immutable collections from Clojure
  • An immutable Transformation Builder which is baked into every collection interface.
  • A tiny domain-specific language of brief helper functions: vec(), tup(), map(), set(), and xform()
  • Simplified functional interfaces that wrap checked exceptions
Also, there are experimental features you may find useful:
  • Unmodifiable interfaces which deprecate mutator methods and throw exceptions to retrofit legacy code and catch errors in your IDE instead of at runtime.
  • An Equator and ComparisonContext which work like `java.util.Comparator`, but for hash-based collections.
  • Memoization for functions
